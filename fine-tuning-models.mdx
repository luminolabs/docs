---
title: "Fine Tuning Models"
description: "On the Lumino Dashboard UI, go to the Fine-tuning tab and click on Create Job."
---

1. Enter the name of the job
2. Select the base model you would like to fine-tune
3. Select the dataset uploaded in the previous step
4. Select a random seed
5. Choose Hyperparamter settings
    a. Number of Epochs
    b. Batch Size
    c. Learning Rate
6. Click on create-finetuning job
