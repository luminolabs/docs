Follow the detailed instructions below to get started quickly.

## Account Setup
### Step 1: Sign Up on Lumino Platform

Visit Our Website: Navigate to Lumino AI.

**Create an Account**

1. Click on the Sign Up button located at the top-right corner of the homepage.
2. Choose your preferred method to sign up:
  - Email: Enter your email address and create a secure password.
  - Google Account: Click on the Google icon to sign up using your Google
  credentials.
  - Microsoft Account: Click on the Microsoft icon to sign up using your Microsoft
  credentials.

3. Upon successful registration, you will receive a confirmation email. Click the link in the
email to verify your account.

### Step 2: Generate an API Key

Navigate to Settings and then API Keys to create an API Key

1. Provide a name or description for your API key to help you remember its purpose.

2. Select Expiration Date &amp; Time for the API Key &amp; Create API Key
3. Save the API key
