title: "Error Codes"
---

List of error codes & causes


| Code                      | Cause                                |
|----------------------------|--------------------------------------|
| 400 - Invalid Request       | Misconfigured request                |
| 401 - Authentication Error  | Missing or Invalid API Key           |
| 403 - Forbidden           |                                      |
| 404 - Not Found             | Invalid Endpoint URL                 |
| 422 - Validation Errors     | Expiration date must be in future    |
| 500 - Server Error          | Unknown server error                 |
